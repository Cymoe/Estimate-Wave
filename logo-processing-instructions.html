Logo Processing Instructions for Estimate Wave

Standardization Requirements:

  1. Dimensions: Maximum 200px width × 80px height (maintain aspect ratio)
  2. Background: Transparent PNG format
  3. Color Treatment: Original colors (CSS will apply grayscale filter)
  4. File Naming: Use lowercase with hyphens (e.g., campos-family-services.png)

Processing Steps:

  1. Open each logo in an image editor (Photoshop, GIMP, or online tool like remove.bg)
  2. Remove any colored backgrounds (especially the black background from Basin GM)
  3. Resize to fit within 200×80px while maintaining aspect ratio
  4. Save as PNG with transparent background
  5. Name files according to the convention above
  6. Place processed files in: /images/client-logos/

CSS Styling Applied:

The website will automatically apply these effects to all logos:

  • Grayscale filter (removes color for consistency)
  • Slight transparency (70% opacity)
  • Color and full opacity on hover
  • Smooth transitions for professional feel
